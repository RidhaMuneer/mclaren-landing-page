# McLaren Landing Page

This is a test project that was requested as part of a job interview. It is a landing page for McLaren, built to showcase front-end skills: recreating a polished, brand-appropriate design with a hero section, navigation with dropdown menus, and a car showcase featuring models such as the Artura, GT, 720S, and 765LT.

The project is built with [Next.js](https://nextjs.org) (App Router), React 19, and TypeScript, and was bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Project Structure

- `src/app` — App Router entry point, layout, and global styles
- `src/components` — UI components (hero section, navigation, dropdown, car showcase)
- `public` — car images, logo, and hero media assets
